# 开源实践1：个人博客搭建

## 一、实验目的
本实验通过 GitHub Pages 与 GitHub Actions 搭建个人博客系统，掌握开源仓库管理与自动化部署流程。

## 二、实验过程
1. 创建 GitHub 公开仓库
2. 在仓库中添加 index.md 博客文件
3. 启用 GitHub Pages 服务
4. 通过 GitHub Actions 自动部署网站
5. 成功访问生成的博客网站

## 三、实验结果
成功部署个人博客网站，并可以正常访问页面内容。

## 四、总结
通过本实验，初步掌握了 GitHub Pages 的使用方法以及 Markdown 博客的构建方式。
